# JBS10 SimpleScripts

100 focused executable scripts for the JBS-10 synchronous iterator foundation.

Covered areas include the `Iterator` global, `%IteratorPrototype%`, array and string iterator prototypes, inherited `@@iterator`, `@@toStringTag`, iterator internal-state visibility, `for...of`, `Array.from`, `Object.fromEntries`, iterator result objects, iterator closing, and wrong-receiver checks.
